> A supercharged git repository manager.

# Fonctionnalités

- cloner un repo
- lister les repos sur la machine
- trier les repos
- taguer les repos
  - comportement spécifique
- informations sur les repos (GitStatus Surcharged)
  - summary
  - release
  - last commit
  - readme
  - location
- mettre à jour un repos (tirer une releaser si dispo)
- cd repos
- commit repos
